## Software Requirements

R version 4.4.3 or higher and JAGS version 4.3.2 or higher. JAGS must be installed separately from https://mcmc-jags.sourceforge.io

To install all required R packages run:

install.packages(c("MixSIAR", "tidyverse", "R2jags", "rjags", "geometry", "sp"))

## Analysis Workflow

Scripts should be run in the following order:

1. R/1_MixSIAR.R
Runs three Bayesian stable isotope mixing model scenarios using MixSIAR. S1 uses an uninformed prior with no concentration dependence. S2 uses an uninformed prior with concentration dependence. S3 uses an informed prior with concentration dependence. Outputs: All_Three_Scenarios_Complete.csv, Posterior_Boxplots_AllSites.pdf, Posterior_Boxplots_AllSites.png.

2. R/2_Metrics.R
Calculates isospace metrics including convex hull area, isotopic ranges, and trophic position estimates, and produces the enhanced isotope biplot. Outputs: Consumer_Isotope_Summary_by_Site.csv, Source_Values_TDF_Corrected.csv, Trophic_Position_by_Site.csv, Isospace_Summary_Metrics.csv, Isospace_Plot_Enhanced.pdf, Isospace_Plot_Enhanced.png.

3. R/3_TieszenEtAl_comparison.R
Compares isotope values from this study to Tieszen et al. reference values using mean differences and one-sample t-tests. Outputs: Tieszen_Mean_Differences.csv, Tieszen_TTest_Results.csv.

4. R/4_Co-Ap_Diachronic.R
Performs collagen–apatite spacing analysis, marine dietary contribution estimates, and diachronic comparisons by time period. Outputs: human_isotopes_clean_filtered.csv, Collagen_Apatite_Spacing_Summary.csv, Marine_Contribution_from_Apatite.csv, Collagen_vs_Apatite_Comparison.csv, Spacing_ANOVA_by_Period.csv.

## Reproducibility

All scripts use setwd() pointing to the repository root. Before running, update the setwd() line at the top of each script to match where you have cloned the repository on your machine, for example setwd("~/Documents/paleodietThesisData"). All file paths are relative to the repository root so no other path changes are needed.

## Notes on Model Objects

The .RData files containing fitted MixSIAR model objects are not included in this repository due to file size constraints. To reproduce them run R/01_MixSIAR_Three_Scenarios.R in full. Note that the very long MCMC chains may take several hours to complete.
